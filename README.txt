Una Pokedex es una herramienta esencial para cualquier entrenador Pokémon que quiera conocer información sobre diferentes tipos de Pokémon. Este proyecto es una Pokedex web desarrollada por Iamsebas376, que te permite explorar y aprender sobre los Pokémon.

## Características

- Lista de Pokémon con detalles como nombre, número y tipo.
- Diseño atractivo y responsivo.
- Uso de colores representativos de los tipos de Pokémon.
- Fácil navegación y búsqueda de Pokémon.

## Cómo se realizó la aplicación

Esta aplicación de Pokedex se construyó utilizando una combinación de tecnologías web estándar, datos de Pokémon de la PokeAPI y personalización de diseño. Aquí se detalla cómo se realizó:

- **HTML5**: Se utilizó HTML5 para la estructura básica de la página, incluyendo la definición de elementos como encabezados, imágenes y enlaces.

- **CSS3**: La aplicación se estilizó y diseñó utilizando CSS3 para crear una interfaz atractiva y responsiva. Se aplicaron estilos a los contenedores de Pokémon, a las imágenes y a otros elementos para lograr un aspecto agradable.

- **JavaScript**: JavaScript se utilizó para la lógica de la aplicación. Se accedió a la PokeAPI para obtener datos de Pokémon y se crearon tarjetas de Pokémon dinámicamente.

- **PokeAPI**: La PokeAPI proporcionó datos esenciales sobre los Pokémon, incluyendo nombres, números, tipos y sprites. Estos datos se obtuvieron mediante solicitudes HTTP y se integraron en la aplicación para mostrar información detallada sobre cada Pokémon.

- **Google Fonts**: Se incorporó la fuente "Lato" de Google Fonts para mejorar la estilización de texto en la aplicación.

La aplicación se desarrolló con un enfoque en la simplicidad y la facilidad de uso, permitiendo a los usuarios explorar y aprender sobre los Pokémon de manera intuitiva. Se espera que esta Pokedex sea tanto informativa como entretenida para los amantes de Pokémon.

Si deseas obtener más detalles técnicos sobre la implementación, puedes explorar el código fuente en este repositorio.
